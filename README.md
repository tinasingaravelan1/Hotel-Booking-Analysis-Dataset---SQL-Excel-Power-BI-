# Hotel Booking Analysis Dataset 🏨

Project Overview:
Analyzed 500+ hotel booking records across 15 countries to uncover revenue patterns, guest behavior and booking channel performance using SQL, Excel and Power BI.

🔍 Key Insights from the Dashboard: 

1. Data Cleaning in Excel 
2. AVERAGEIF : Avg Rating per Room Type and COUNTIF: Most Popular Booking Channel
3. Conditional Formatting: Top Spending Bookings and Pivot Tables and Pivot Charts in Excel
4. OTA channels (MakeMyTrip & Booking.com) drive the highest revenue India and Singapore are top revenue generating countries 
6. August recorded peak monthly revenue 
7. Average guest rating of 3.97 out of 5 
8. Business and Couple guests contribute the most bookings

Tools Used
- SQL - Data extraction, validation and channel level 
  insights using GROUP BY, JOINS and aggregations
- Microsoft Excel - SUMIF, AVERAGEIF, COUNTIF, 
  Pivot Tables, Conditional Formatting
- Power BI - Interactive dashboard, slicers, map visuals
- DAX - 10+ measures including Time Intelligence, 
  Revenue Contribution % and Booking Channel Rank

Dashboard Features
- Revenue by Country - World map with bubble sizes 
  representing revenue per country
- Monthly Revenue Trend - Line chart showing Jan to Dec 
  revenue fluctuations with peak identification
- Revenue by Booking Channel - Sorted bar chart showing 
  OTA vs Direct vs Corporate performance
- Avg Guest Rating Card - Dynamic card updating with 
  Room Type and Guest Type slicers
- Avg Revenue Per Guest Card - DAX powered KPI card

Data Validation Steps
- Checked for duplicates - none found
- Verified date formats and converted to proper date type
- Null value analysis across all 13 columns
- Confirmed 100% data integrity before analysis

DAX Measures Built
- Total Revenue
- Total Bookings
- Avg Rating
- Avg Nights Stayed
- Avg Bill Per Booking
- Monthly Revenue
- Revenue Contribution %
- Booking Channel Rank
- Avg Revenue Per Guest
- Guest Segment

Excel Analysis Performed
- AVERAGEIF - Average rating per room type
- COUNTIF - Most popular booking channel
- SUMIF - Revenue by room category
- Conditional Formatting - Top 20 spending bookings
- Pivot Tables - Revenue by country and room type
- Pivot Chart - Monthly booking trends

 How I used AI in this project?

Leveraged Claude AI as a productivity tool to speed up my workflow from generating a realistic dataset to troubleshooting errors in real time. This allowed me to focus more on the actual analysis, interpretation and visualization rather than spending time on manual data creation and debugging. All business insights, dashboard design decisions and analytical thinking were entirely done from my end !
